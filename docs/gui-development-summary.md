## 💡 核心特性

### 1. 前后端通信
```javascript
// 前端调用
const results = await invoke('scan_all_caches');

// Rust 处理
#[tauri::command]
async fn scan_all_caches(
    state: State<'_, AppState>,
    window: tauri::Window,
) -> Result<Vec<ScanResultItem>, String> {
    // 实现扫描逻辑
}
```

### 2. 实时进度更新
```rust
// 发送进度事件
window.emit("scan-progress", &progress)?;

// 前端监听
await listen('scan-progress', (event) => {
    updateProgressBar(event.payload);
});
```

### 3. 选择性清理
```javascript
// 前端选中
const selectedItems = scanResults.filter(r => r.selected);

// 调用清理
await invoke('clean_selected_caches', { selectedItems });
```
