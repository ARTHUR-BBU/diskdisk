# 更新日志

## [0.2.0] - 2026-03-17

### 新增 ✨

- **扩展缓存类型支持：30 → 38 种**
  - 新增笔记和文档类：Notion 笔记
  - 新增视频会议类：Zoom 视频会议
  - 新增安全软件类：360 安全软件、360浏览器
  - 新增 SDK 和工具：SecretSDK 缓存
  - 新增交易平台：FTNN 交易所缓存
  - 新增 OCR 工具：2345OCR 缓存
  - 新增 Python 环境缓存

### 改进 🚀

- 完善所有缓存类型的环境变量展开支持
- 优化缓存路径配置，覆盖更多实际应用场景
- 更新 desktop-gui 解析逻辑，支持所有 38 种缓存类型

### 扫描能力提升 📊

**扩展前扫描结果：**
- 总计：约 2.55 GB（93,293 文件）

**扩展后扫描结果：**
- 总计：5.44 GB（46,536 文件）
- 仅新增缓存类型就有：**4.86 GB**
  - Notion 笔记：1.88 GB (16,070 文件)
  - SecretSDK 缓存：829.15 MB (2,829 文件)
  - FTNN 交易所：645.92 MB (5,524 文件)
  - Zoom 会议：571.58 MB (970 文件)
  - 360 安全软件：572.45 MB (5,235 文件)
  - Python 环境：313.26 MB (8,851 文件)
  - 360浏览器：270.18 MB (1,870 文件)
  - 2345OCR：240.73 MB (1,022 文件)

### 技术细节 🔧

- 修改文件：
  - `crates/core/src/cache_types.rs` - 添加 8 个新枚举值
  - `crates/core/src/scanner.rs` - 添加 8 个新缓存路径配置
  - `desktop-gui/src/lib.rs` - 更新 parse_cache_type() 函数

- 所有代码通过 cargo check 验证
- 核心库和 CLI 编译成功
